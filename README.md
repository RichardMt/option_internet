# Geo Image Search

A content-based retrieval system with added geolocalization 

## Objective

The objective of this project is to propose a search engine for live images based on raw streamed image content coupled with 
geolocalization data for mobile devices

The system must offer :
* Realtime geolocalization of the user device
* Realtime acquisition of images from user's camera
* Interoperation with a server-side similarity search engine
* Display of most similar images and places
